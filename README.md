### ✨ Hub Features
- 🔍 **Smart Search** — Filter projects by name, category, or tech in real-time
- 📂 **Auto-Detection** — New `XX.ProjectName` folders are detected automatically by a Node.js watcher

## ⚡ Quick Start

Get the hub running locally in under 60 seconds:

### Prerequisites

- [Node.js](https://nodejs.org/) v16+ installed
- Git

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Mr-S-U-D-O/javascriptProjectBasedLearning.git

# 2. Navigate into the project
cd javascriptProjectBasedLearning

# 3. Run the master command
npm start
```

> 💡 **`npm start`** does everything: spins up a local dev server, builds the Hub data from all project folders, and starts a file watcher that auto-detects new projects as you add them.

### Individual Scripts

| Command | Description |
|:--------|:------------|
| `npm start` | 🚀 **Master command** — Server + Build + Watch |
| `npm run update` | 🔄 Manually rebuild the `projects.json` data |
| `npm run watch` | 👁️ Watch for new project folders only |

### Adding a New Project

```bash
# Simply create a new numbered folder:
mkdir "30.MyAwesomeProject"

# Add your files inside:
#   30.MyAwesomeProject/
#   ├── index.html
#   ├── style.css
#   └── script.js

# The watcher will auto-detect it and add it to the Hub! ✨
```

## 🚢 Deployment

The Hub is automatically deployed to **GitHub Pages** on every push to `main`:

```yaml
# .github/workflows/deploy.yml
on:
  push:
    branches: [main]

# Deploys the hub/ directory as the live site
```
