# ⚠️ CRITICAL WARNING: DO NOT CLONE
**This repository is strictly for in-app contribution only.** 
Do NOT clone this repository to your local machine. All development and contributions must be performed directly through the **[S.U.D.O Project Hub IDE](https://mr-s-u-d-o.github.io/javascriptProjectBasedLearning/hub/)**. 

*Any local clones may be outdated and will not sync properly with the Hub's automated deployment engine.*

## 🚀 How It Works

This repository is powered by a custom Node.js automation engine.
1.  **Open the Hub**: Navigate to the live [Project Hub](https://mr-s-u-d-o.github.io/javascriptProjectBasedLearning/hub/).
2.  **Use the IDE**: Use the built-in VS Code-style IDE to create or edit projects.
3.  **Submit PR**: Submit your changes directly from the browser.

## 🤝 How to Contribute

Contributions are welcome! Please follow these steps **strictly**:

1. Open the [S.U.D.O Project Hub](https://mr-s-u-d-o.github.io/javascriptProjectBasedLearning/hub/).
2. Click on **CONTRIBUTE** in the dashboard.
3. Use the **In-App IDE** to build your project.
4. Click **SUBMIT PR** when finished.
