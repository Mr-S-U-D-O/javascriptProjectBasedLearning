# Javascript Project-Based Learning Journey

Welcome to my frontend learning hub. This repository is a collection of side projects I built while mastering HTML, CSS, and Javascript. Each project represents a step forward in my understanding of modern web development, ranging from simple UI components to complex interactive applications.

## 🚀 The Hub

I've created a central **[Project Hub](./hub/index.html)** where you can navigate through all the projects.

### How it works:
- **Automatic Detection**: When I add a new project subdirectory (following the `XX.ProjectName` naming convention), a custom Node.js script automatically detects it.
- **Similarity-Based Grouping**: Projects are grouped by category (e.g., Generators, Visual Effects, Dashboards) based on their names.
- **Sleek Aesthetic**: The hub uses a high-contrast, border-heavy "thinline" aesthetic for a premium developer experience.

---

## 🛠️ Project Categories

- **Generators**: Tools for creating CSS effects (Clip Paths, Gradients, Shadows).
- **Dashboards**: Administrative and data-heavy interfaces.
- **Visual Effects**: Experiments with animations, transitions, and layout shifts.
- **Interactives**: Games and interactive UI elements.
- **UI Components**: Specialized kits and reusable elements.

## 🏗️ Technical Details

### Automation Script
The hub's data is maintained by `scripts/update-hub.js`.

**To update the hub manually:**
```bash
node scripts/update-hub.js
```

### Aesthetic Blueprint
- **Colors**: Neo-Matrix Green on Black.
- **Borders**: 2px solid accents.
- **Typography**: Space Mono for that terminal-inspired feel.
