S.U.D.O SYSTEM MANUAL
===================
NAME
       sudo-hub - Super thinlined project navigation dashboard

DESCRIPTION
       This internal terminal hub provides direct access to all tracked 
       javascript learning projects. Use standard commands to interact 
       with the interface.

COMMANDS
       help         - Show all available terminal commands.
       stats        - View system analytics and language metrics.
       history      - Pull live Git Commit logs from the origin API.
       guide        - Display this manual page.
       contribute   - Launch the in-app IDE. Write a project and submit a PR.
       contributors - View the GitHub contributor wall for this repo.
       clear        - Reset the terminal and return to the project listing.

       [ FUTURE ENHANCEMENTS ]
       socials, blog, feedback, settings, arcade, themes, resources, 
       license, support, changelog.

HOW TO CONTRIBUTE (STEP-BY-STEP)
       1. IN-APP IDE: Type 'contribute' in the terminal to launch the 
          built-in code editor.
       2. TEMPLATE: Choose a starter template (Calculator, Todo, etc.) 
          or start from a blank project.
       3. DEVELOP: Write your HTML, CSS, and JS. Use the 'Live Preview' 
          pane to see your changes in real-time.
       4. CHECKLIST: Click 'Submit PR' and complete the quality 
          checklist (ensure README.md is included!).
       5. ZIP & UPLOAD: Enter your project title and click 'Package'. 
          A .zip file will download, and you'll be redirected to 
          GitHub's upload page.
       6. PR: Drag your .zip file (or extracted folder) into the 
          GitHub upload area and create a Pull Request.

CSS TRICKS LEARNED
       - Glassmorphism: Use backdrop-filter: blur(10px) combined with
         a semi-transparent rgba() background.
       - Scanlines: Repeating linear gradients with fixed background 
         attachment and simple vertical CSS translation.
       - Multi-column Masonry: Using pure CSS column-count for 
         staggered grid perfection. 

JAVASCRIPT TRICKS LEARNED
       - Web Audio API is entirely self-contained; oscillators can
         generate mechanical synths without .mp3s.
       - Debounced Watchers: Using Node's fs.watch with a 1000ms 
         setTimeout debounce prevents double-execution.
       - Event Delegation: Attaching listeners to a parent instead 
         of individual DOM children saves memory and covers dynamic elements.

MAINTENANCE
       To start the unified development server:
       $ npm start

       Report bugs to the git repository's issue tracker.
